The chart shows the percentage of each sentiment extracted from the comments. Each bar represents one video or group of videos.

Sentiment scores are calculated with three sentiment analysis tools (AFINN, NLTK, and TextBlob). Each tool returns a polarity value when the input text is evaluated as showing positive or negative sentiment. They return neutral when the text contains no sentiment bearing word or when negative and positive words are perfectly balanced. "Mixed sentiment" means that one comment was detected as both positive and negative by different tools. Non-English comments are not assessed by the tools and are shown as N/A (not available).
